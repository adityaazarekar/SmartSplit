# SmartSplit - Advanced Expense Manager 🚀

SmartSplit is a professional-grade desktop application for managing group expenses with ease. Built with Java Swing and custom modern UI components, it features dynamic currency conversion, role-based interaction, and slick animated themes.

## ✨ Key Features

- **Multi-Currency System:** Support for INR, USD, EUR, GBP, and JPY with dynamic symbol formatting.
- **Interactive UI:** Animated glass pane background with mouse-tracking spotlights and context-aware "aurora" themes.
- **Expense Categorization:** Categorize spending (Food, Travel, etc.) and view visual monthly analytics.
- **Member Management:** Add teammates, set custom profile pictures, and track individual balances.
- **Receipt Previews:** Attach and view proof of payment for any expense.
- **Persistence:** Automatically saves your groups and expenses to a local workspace.

## 🚀 Getting Started

### Prerequisites
- **Java JDK (Version 17 or higher recommended)**

### Installation & Run
1. **Clone the repository:**
   ```bash
   git clone https://github.com/adityaazarekar/SmartSplit.git
   cd SmartSplit
   ```
2. **Run the application:**
   Simply double-click the `SmartSplit.bat` file in the root directory (on Windows), or run it from the terminal:
   ```powershell
   .\SmartSplit.bat
   ```

## 🛠 Compilation
The project includes a `SmartSplit.bat` script that handles:
- Locating the Java compiler
- Compiling all source files into the `out` directory
- Launching the application with the correct classpath

## 🎨 Professional Design
Unlike standard Swing applications, SmartSplit uses a centralized `UITheme` for:
- Pill-shaped buttons with hover effects
- Glassmorphic card panels
- Custom scrollbars and focus highlights
- Scalable vector-like typography
